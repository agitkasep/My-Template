# Quick Auth Template (React)

Template dasar untuk sistem autentikasi (Login/Logout) menggunakan React Hooks. Dibuat untuk demonstrasi kecepatan koding dan efisiensi struktur komponen.

## Fitur:
- Login Logic (React Hooks)
- Protected Routing
- UI Responsive (Tailwind CSS)

## Cara Penggunaan:
1. Copy kode ke project React Anda.
2. Pastikan Tailwind CSS sudah terpasang.
3. Jalankan `npm start`.
